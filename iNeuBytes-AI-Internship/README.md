# iNeuBytes Artificial Intelligence Virtual Internship

## Task 1 — Computer Vision using CNN Models

This repository implements a reproducible TensorFlow/Keras study on CIFAR-10: an augmentation-free AlexNet-inspired baseline, controlled one-variable experiments, and a final customized CNN. It automatically produces the master experiment table, learning curves, architecture summaries, classification metrics, confusion matrices, and most-confused class pairs.

### Reproducibility

- Seed: `20726` for Python, NumPy, and TensorFlow
- Frozen split: 45,000 train / 5,000 validation / 10,000 official test images
- Fixed budget: 25 epochs and batch size 128
- One-variable experiment definitions: `configs/experiments.yaml`
- CIFAR-10 test data is never used for model selection

### Experiment design

| Group | Controlled comparisons |
|---|---|
| Regularization | Baseline vs dropout vs batch normalization vs L2 |
| Augmentation | Baseline vs light vs moderate vs aggressive |
| Optimization | Adam vs SGD+momentum vs RMSprop; learning rates 0.01, 0.001, 0.0001 |
| Architecture | Baseline vs deeper CNN vs 5×5 first kernels |
| Final | Evidence-supported winning techniques combined |

### Integrity note

Results are intentionally generated by the training run and are not fabricated or hard-coded. Hardware and TensorFlow versions should be recorded in the report because training time can vary across systems.
